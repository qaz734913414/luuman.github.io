rem是什么？

rem（font size of the root element）是指相对于根元素的字体大小的单位。
em（font size of the element）是指相对于父元素的字体大小的单位。

区别：它们之间其实很相似，只不过一个计算的规则是依赖根元素一个是依赖父元素计算。
